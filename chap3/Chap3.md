## 練習問題3.2

1. 変数valueがnullの場合は「規定値」、そうでなければvalueの値を出力するようなコードを条件演算子、null合体演算子それぞれを利用したパターンで書いてみましょう。

```
value = 3.2;
def = 0;

// 条件演算子
Console.WriteLine( value == null ? def : value );

// null 合体演算子
Console.WriteLine( value ?? def );

```
